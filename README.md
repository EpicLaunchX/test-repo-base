## Setup
### Project dependencies:
 * Python 3.10
 * [GNU make](https://www.gnu.org/software/make/)
 * [Virtualenv](https://virtualenv.pypa.io/en/latest/)

Create virtualenv in the project root directory named ".venv"

### Commands

`make install-flit`

`make install-dev`

`make run` or `make test`
